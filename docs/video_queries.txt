Video dataset ferrara con scalar temperatura (acquisizione con Acquisition_tag = “day”)
PREFIX base:<http://www.semanticweb.org/mcodi/ontologies/2024/3/Urban_Ontology#>
SELECT DISTINCT ?s ?x 
FROM <http://localhost:8890/Ferrara>
WHERE {
    ?acq base:Contains_data_relative_to ?s.
    ?acq base:Temperature ?x.
    ?acq base:Acquisition_tag ?tag.
    FILTER(?tag = "day")
}


Video dataset ferrara con scalar permeability to water (è attribuita ai materiali, i quali a loro volta sono connessi ad acquisizioni e poi a punti. Mi sembra di averti mandato gli esempi per query del genere a suo tempo)
PREFIX base:<http://www.semanticweb.org/mcodi/ontologies/2024/3/Urban_Ontology#>

FROM <http://localhost:8890/Ferrara>
WHERE {
    ?acq base:Is_a_point_made_of ?mat.
    ?mat base:Permeability_To_Water ?x.
    ?acq base:Contains_data_relative_to ?s.
}
